# IT Helpdesk & Asset Management System

این پروژه یک سیستم جامع برای مدیریت درخواست‌های IT (تیکتینگ)، 
مدیریت تجهیزات (Asset Management)، گزارش‌گیری و ارتباط آنلاین بین کارمندان و واحد IT است.  

## ✨ قابلیت‌ها
- ثبت و پیگیری تیکت‌های پشتیبانی
- امکان چت با تیم IT
- مدیریت تجهیزات تخصیص داده شده به کارمندان
- داشبورد مدیریتی برای گزارش‌ها و KPI
- ثبت گزارش کار کارشناسان IT
- سیستم امتیازدهی و بازخورد کاربران

## 🚀 آینده
- یکپارچه‌سازی با Active Directory
- هوش مصنوعی برای پیشنهاد راه‌حل
- مانیتورینگ خودکار تجهیزات
- اپلیکیشن موبایل (PWA)
